# niko.github.io

Static bilingual site for a GitHub Pages user repository named `niko.github.io`.

## Structure

- `index.html`: page shell
- `assets/site.css`: visual design and responsive layout
- `assets/site.js`: bilingual content data and rendering logic
- `.nojekyll`: keeps GitHub Pages from applying Jekyll processing

## Publish

1. Create a GitHub repository named `niko.github.io`.
2. Push these files to the default branch of that repository.
3. GitHub Pages will serve the site directly from the repository root.

## Notes

- Language selection is stored in `localStorage`.
- The timeline content is adapted from the provided HTML source and reorganized into a data-driven bilingual presentation.
